## 二、申请步骤

### 第1步：注册登录中国版权保护中心

- 网址：**https://www.ccopyright.com.cn/**
- 点击"软件登记"→"个人注册"
- 用身份证实名认证

### 第2步：填写申请表

登录后选择"软件著作权登记"，填写：

| 字段 | 填写内容 |
|------|---------|
| 软件名称 | 筮问·SHI WEN（CYBER I-CHING） |
| 版本号 | V2.2.0 |
| 开发完成日期 | 2026年3月 |
| 发表日期 | 2026年3月 |
| 著作权人 | [你的真实姓名] |
| 开发方式 | 独立开发 |
| 权利取得方式 | 原始取得 |
| 软件用途 | 文化娱乐 |
| 编程语言 | Python |
| 源代码行数 | 约 3000 行 |

### 第3步：上传材料

- 上传软件说明书（PDF格式）
- 上传源代码摘录（PDF格式，前30页+后30页）
- 上传身份证正反面扫描件

### 第4步：缴费

- 个人申请费用：**300元**（官方标准）
- 支持网上支付

### 第5步：等待审核

- 一般审核周期：**15-30个工作日**
- 审核通过后邮寄证书到你的地址

| 材料 | 要求 | 状态 |
