## 一、软件简介

筮问·SHI WEN 是一款以中国传统易经文化为核心的数字化占卜软件，融合赛博朋克视觉风格，将六十四卦、三百八十四爻的古典哲学内容以现代交互方式呈现。软件提供铜钱摇卦、随机起卦两种方式，自动计算卦象并输出完整的卦辞、爻辞及白话解析，帮助用户借助传统智慧进行自我反思与决策参考。

软件采用赛博朋克（Cyberpunk）美学风格设计，以深色背景配合霓虹色彩（青色 #00F5FF、粉色 #FF2D9B、紫色 #B44FFF）营造独特的视觉体验，是一款将东方古典文化与现代科技美学深度融合的创新型文化软件。

## 二、软件功能描述

### 2.1 核心功能

#### 2.1.1 铜钱摇卦（手动起卦）
用户通过点击"投掷铜钱"按钮，模拟传统三枚铜钱摇卦过程：
- 第一次投掷：生成下卦（初爻、二爻、三爻）
- 第二次投掷：生成上卦（四爻、五爻、上爻）
- 每次投掷随机生成三枚铜钱的正反面，以三枚铜钱总点数判定爻的阴阳属性
- 阳爻以黄色实线显示，阴爻以青色断线显示，视觉区分明确

#### 2.1.2 一键起卦（自动随机）
用户点击"一键起卦"按钮，软件自动完成两次投掷，即时生成完整六爻卦象，适合快速占卜场景。

#### 2.1.3 卦象识别与查询
软件内置完整的六十四卦数据库，通过六爻二进制编码（下卦三位 + 上卦三位）唯一定位卦象：
- 自动识别本卦名称、卦序编号、卦位描述（上下卦组合）
- 展示对应卦象符号（Unicode 卦符）
- 显示完整卦辞原文及白话解析

#### 2.1.4 爻辞逐爻解读
软件内置三百八十四条爻辞数据（六十四卦 × 六爻），对每一爻提供：
- 爻辞原文（古典文言文）
- 白话释义（现代汉语解析）
- 动爻标注（区分老阴、老阳变爻）

#### 2.1.5 卦象解读展示
结果界面以结构化方式展示：
- 卦序 + 卦名 + 卦象符号
- 卦位描述（如"坤上震下"）
- 卦辞原文
- 卦义解析（历史文化背景）
- 白话析义（现代语境解读）
- 六爻逐爻解读（含动爻变卦说明）

### 2.2 界面功能

#### 2.2.1 主界面布局
软件主界面分为三个功能区域：
- **左列**：铜钱投掷操作区（固定宽度360px），包含轮次显示、三枚铜钱动画、操作按钮、状态提示
- **右上区**：六爻状态展示区 + 卦象预览区，实时显示当前爻的阴阳状态
- **右下区**：卦象解读滚动文本区，展示完整解读内容

#### 2.2.2 铜钱动画效果
三枚铜钱支持旋转动画效果，投掷时呈现翻转动画，显示正（阳）反（阴）面，增强互动体验感。

#### 2.2.3 实时爻线绘制
六爻状态区采用 Canvas 绘图：
- 阳爻：黄色实线，贯穿整个宽度
- 阴爻：青色断线，两段等长，中间空隙精确居中
- 待定状态：虚线显示，颜色置灰

#### 2.2.4 赛博朋克视觉主题
- 深色背景主题（背景色 #0A0A0F）
- 霓虹色彩体系（青色、粉色、紫色、绿色）
- 等宽字体（Consolas）与中文字体（Microsoft YaHei）混排
- 边框发光效果模拟

### 2.3 数据管理

#### 2.3.1 六十四卦元数据（iching_data.py）
- 每卦包含：卦序、卦名、卦位描述、卦辞、卦义、白话解析
- 以六位二进制元组为唯一键（保证零冲突）
- 提供 get_hexagram() 查询接口

#### 2.3.2 三百八十四爻辞数据（yaoci_data.py）
- 每卦六爻完整收录，共384条爻辞
- 每条爻辞包含：原文 + 白话释义
- 提供 get_yaoci() 查询接口

## 四、软件操作说明

### 4.1 起卦流程

**手动起卦（推荐）：**
1. 启动软件，进入主界面
2. 点击粉色按钮"投 掷 铜 钱"
3. 三枚铜钱完成翻转动画，下卦（初/二/三爻）自动生成
4. 再次点击"投掷第二次"
5. 上卦（四/五/上爻）生成，六爻完整呈现
6. 卦象解读自动展示在右侧/下方区域
7. 点击"重 置"可重新起卦

**一键起卦：**
1. 点击"一键起卦"按钮
2. 软件自动完成两次投掷
3. 直接进入解读界面

### 4.2 查看解读
- 卦象基本信息显示在右上角卦象预览区
- 详细解读在下方滚动文本区，可上下滑动查看
- 解读内容包含：卦辞 → 卦义 → 白话析义 → 逐爻解读

```
    ├── main.py          # Android 版主程序
    └── buildozer.spec   # Android 打包配置
```

### 5.2 核心算法

**卦象计算算法：**
1. 三枚铜钱投掷结果：正面=3，反面=2
2. 三枚总点数：6（老阴）、7（少阳）、8（少阴）、9（老阳）
3. 奇数为阳爻，偶数为阴爻
4. 六爻依次编码为二进制元组
5. 以（bit0,bit1,bit2,bit3,bit4,bit5）查询卦象字典

**唯一键编码规则：**
- bit0~bit2：下卦（初爻→三爻），乾=111，坤=000，震=100，艮=001，巽=011，兑=110，坎=101，离=010
- bit3~bit5：上卦（四爻→上爻），同上规则
- 64个卦象键值唯一，无冲突
